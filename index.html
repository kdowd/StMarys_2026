Basic HTML / CSS Usage:

We'll spend the next 30 mins getting used to VSCode and building a simple web page.

We will create some code to style our webpage. Just so you know some basics...

Doing all of the above will help prepare you for the rest of today.

I will be sharing code with you so you'll need to have some idea where the code goes and how it relates to the page and page style.

Start VSCode Project
  • Create a Folder on your machine and give it a meaningful name - e.g. myProject. Remember where you made the folder.
  • Open VSCode and use "File > Open Folder" and navigate to the folder you just created. Or, drag and drop the new folder into the current VSCode window. Either of the 2 methods will create a new project in VSCode.
  • In VSCode your working files are on the left of the window - "Explorer". Using the explorer create these 3 files.
    1. index.html
    2. shared.css
    3. navigation.css

File and Folder names are always CaSe-SeNsiTive.

 Updated index.html:

  1. Paste the following into index.html

    <!doctype html> <html lang="en"> <head> <meta charset="UTF-8" /> <meta name="viewport" content="width=device-width, initial-scale=1.0" /> <link rel="stylesheet" href="./shared.css" /> <link rel="stylesheet" href="./navigation.css" /> <title>Document</title> </head> <body> <div class="container"> <!-- header start--> <header> your navigation here. </header> <!-- main content start --> <main class="content"> <p>your main content here.</p> </main> <!-- footer start --> <footer> <div> <p>your footer details here.</p> </div> <div> <p class="copyright">&copy; 2026</p> </div> </footer> </div> </body> </html>
